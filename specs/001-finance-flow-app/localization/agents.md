# Agents

## Localization Agent

### Purpose

Implement multilingual support for FinanceFlow.

### Responsibilities

* Create translation infrastructure
* Manage translation dictionaries
* Ensure fallback to English
* Maintain consistency across languages

### Inputs

* UI text
* Translation dictionaries

### Outputs

* Localized interface
* Translation mappings

---

## UI Integration Agent

### Purpose

Integrate localization into Streamlit pages.

### Responsibilities

* Replace hardcoded strings
* Connect translation helper functions
* Maintain existing UI behavior

### Inputs

* Existing Streamlit pages
* Translation utilities

### Outputs

* Language-aware UI components

---

## QA Agent

### Purpose

Validate localization quality.

### Responsibilities

* Verify translation coverage
* Test language switching
* Check fallback behavior
* Detect missing keys

### Outputs

* Validation reports
* Localization fixes
