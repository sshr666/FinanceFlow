# Skills

## Ollama Integration

- Use local Ollama endpoint
- Use requests library
- Use phi3:mini model
- Disable streaming initially

Endpoint:
http://localhost:11434/api/generate

## Prompt Engineering

Generate:
- Spending trends
- Savings observations
- Budget warnings
- Actionable recommendations

Keep responses concise and user-friendly.

## Error Handling

If Ollama is unavailable:
- Show Streamlit warning
- Do not crash application
